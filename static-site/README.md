## How to run

 OS Dependencies:
```
sudo apt install awscli unzip
```

### Download, Install Requirements:

```bash
wget https://releases.hashicorp.com/terraform/0.13.0/terraform_0.13.0_linux_amd64.zip
unzip terraform_0.13.0_linux_amd64.zip
mv terraform /bin/
```

### Run terraform
```
terraform init
terraform plan
terraform apply
```
